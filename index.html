How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Note: This game is a modification from 2048, which you can play via https://gabrielecirulli.github.io/2048/. Why this modification? You may be able to always get to the 4096 tile, but getting to 16384 is harder, so instead of the tedium to set up the 4096 tile, this version lets you start from there.

Created by Frazer Kirkman. You could create your own version by cloning Gabriel's version at https://github.com/gabrielecirulli/2048 or this version at https://github.com/frazer/8192.
